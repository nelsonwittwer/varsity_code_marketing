Introduction to Software Development

Coding in Minecraft helps your child visualize the abstract nature of code and math.

Learn to Code in Minecraft

What better way to learn the fundamentals of coding than by automating the wide world of Minecraft? Learn how to automatically build towering structures and complex landscapes with your very own code.

Build Your own Microcomputer

The Raspberry Pi is a small credit card sized micro computer that runs a flavor of Linux, and is the perfect platform to learn how to code. You will be guided through installing Linux, setting up your coding environment, and installing a special Raspberry Pi edition of Minecraft. Best of all, this Raspberry Pi is yours to take home and work on your own time.

Explore Linux

We will learn and code on the Linux operating system. Linux powers the majority of web severs today and is completley open source, meaning the code is accesible and open to everyone to read and modify. Learning Linux will you prepare you to manage servers of your own and will open a world of open source development tools that software industry uses every day.

Course Format

  • Week 1

    Introduction to your Raspberry Pi and Coding

    Setup your Raspberry Pi, and get a feel for your new coding environment.

  • Week 2

    Introduction to Coding in Minecraft

    Now that we have our computers built, we will explore the possibilities of writing code to build Minecraft structures.

  • Week 3

    Code Logic - Part 1

    We'll learn the basics of conditional code logic to allow us to build more advanced structures.

  • Week 4

    Code Logic - Part 2

    We'll learn how to better compose our code by learning about methods.

  • Week 5

    Code Logic - Part 3

    Extract logic into repetable classes and object oriented programming.

  • Week 6

    Bringing it All Together

    Now that we know the basics, we'll bring it all together and show you how to build some really amazing things in Minecraft.

Frequently Asked Questions

How much does enrollment cost?

$350 for the 6 week course. The Raspberry Pi microcomputer and accessories that your student will be working with is included inthe cost.

Where will classes be held?

The location is still to be determined. Somewhere between South Jordan and Lehi though.

What Pre-requisites Do you Recommend?

The biggested indicator for success is an excitement about the course itself. If they are excited abotu learnign to code, or exploring new possibilities they will be primed to succeed.

A firm foundation in math will also help out. We will be using variables and plotting in three dimension grids. As long as they are ok learning and working in these concepts they will be fine. This is a great way to help them utilize all that math they never thought they'd actually use :)

What kind of out of class support do you offer?

If you are stuck, you are more than welcome to email the course instructor to ask for help during the week.

Can I take the class with my student?

We have a limited number of seats for each class, so if you are interested in participating with your student, you'll need to enroll for two spots.

Additionally, we'll have the lectures online if you'd like to watch and keep up with your student.

I really loved seeing how much progress my student made, do you offer further courses?

That is definitely the plan! Stay tuned for advanced Minecraft classes and classes in robotics, and web development.

What Will My Student Need to Bring each Week?

Just your Raspberry Pi. We will provide everything else.

Will there be Homework?

Yes! Practice makes perfect! We'll have a weekly assignments which will be creative in nature. Each assignment is designed to help yoru student get an understanding of the concepts covered in each class.

Do we get to keep the Raspberry Pi?

Yes! We want to encourage your student to keep learning, long after this course concludes. During the course, we expect your student to use their Raspberry Pi to complete the homework and show off their creations during the start of each class.

How do we use the Raspberry Pi at Home?

The Raspberry Pi comes with an HDMI output. You can hook it up to any TV or Computer Monitor with an HDMI input. If you have a monitor with a DVI input, you could buy a converter and still use it. Contact us and we'll give you pointers on it.

You'll also need a keyboard and mouse. If you don't have an old set laying around, you can pick up both for about $20 on Amazon.

Can We Bring Our Own Computers?

Unfortunately, no. In order to simplify the learning and teaching process, we'd like everyone to work in the same environment. We are really excited about the Raspberry Pi format, as we can also inclucde lessons in computer hardware. Plus, working on a microcomputer is just really cool!

What if We Can't Make one of the Classes?

We will record every class session and post lectures on our website. You'll just have to make sure you are up to date before the next class, as each class builds on the previous class.
